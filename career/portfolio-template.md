## Case Study Examples

### Case Study 1: New Product Launch

```
CASE STUDY 1: Kitchen Product Launch

THE SITUATION
• Product: Silicone Baking Mat Set
• Category: Kitchen
• Monthly Ad Spend: $1,500
• Starting ACoS: N/A (new product)
• Starting Revenue: $0 (just launched)

THE CHALLENGE
• Zero reviews, zero sales history
• High competition in Kitchen category
• Limited budget for launch
• Needed to build organic ranking fast

MY APPROACH
Week 1-2: Optimized listing (7 images, A+ Content, 5 bullets)
Week 3-4: Launched auto + manual exact campaigns ($50/day)
Week 5-8: Harvested 25 converting terms, expanded to phrase
Week 9-12: Added Sponsored Brands video, scaled to $100/day

THE RESULTS
┌─────────────────────────────────────────┐
│ Metric          │ Day 1   │ Day 90      │
├─────────────────┼─────────┼─────────────┤
│ ACoS            │ N/A     │ 28%         │
│ Revenue         │ $0      │ $4,200/mo   │
│ Reviews         │ 0       │ 45          │
│ Organic Rank    │ Page 5  │ Page 1      │
└─────────────────────────────────────────┘

KEY TAKEAWAYS
• Listing optimization before ads is critical
• 90 days minimum for meaningful organic improvement
• Don't scale too fast — let data guide decisions
```

### Case Study 2: Account Restructuring

```
CASE STUDY 2: Supplement Brand Turnaround

THE SITUATION
• Product: Vitamin Supplements
• Category: Health & Grocery
• Monthly Ad Spend: $8,000
• Starting ACoS: 75%
• Starting Revenue: $10,500/month

THE CHALLENGE
• ACoS well above break-even (40%)
• 40+ campaigns with $5/day budgets (too thin)
• No keyword segmentation
• Overlapping campaigns competing against each other

MY APPROACH
Week 1: Full audit — identified 200+ waste keywords
Week 2: Consolidated 40 campaigns → 12 focused campaigns
Week 3-4: Built proper structure (auto → manual harvest)
Week 5-8: Aggressive bid optimization, negated 150+ terms
Week 9-12: Expanded to SB + SD, maintained profitability

THE RESULTS
┌─────────────────────────────────────────┐
│ Metric          │ Before  │ After       │
├─────────────────┼─────────┼─────────────┤
│ ACoS            │ 75%     │ 28%         │
│ Revenue         │ $10,500 │ $18,200/mo  │
│ ROAS            │ 1.3x    │ 3.6x        │
│ Campaigns       │ 40      │ 12          │
│ Time to Manage  │ 20 hrs  │ 8 hrs/week  │
└─────────────────────────────────────────┘

KEY TAKEAWAYS
• More campaigns ≠ better performance
• Consolidation improves efficiency
• Negative keywords are as important as positive ones
```

### Case Study 3: Q4 Seasonal Campaign

```
CASE STUDY 3: Toy Brand Q4 Domination

THE SITUATION
• Product: Educational Toys
• Category: Toys & Games
• Monthly Ad Spend: $5,000 (normal) → $15,000 (Q4)
• Starting ACoS: 32%
• Starting Revenue: $15,000/month

THE CHALLENGE
• Needed to maximize Q4 revenue (50% of annual sales)
• Competitors increasing spend aggressively
• Limited budget compared to big brands
• Had to be strategic, not just outspend

MY APPROACH
October: Built 20 seasonal campaigns, set bid baselines
November 1-15: Ramped spend 20% weekly, gathered data
November 16-24: Aggressive bids (+50%), $500/day budget
Black Friday: Hourly monitoring, real-time adjustments
December: Maintained high bids, harvested winning terms

THE RESULTS
┌─────────────────────────────────────────┐
│ Metric          │ Q4 2025 │ Q4 2026     │
├─────────────────┼─────────┼─────────────┤
│ Revenue         │ $45,000 │ $98,000     │
│ ACoS            │ 35%     │ 22%         │
│ Ad Spend        │ $15,750 │ $21,560     │
│ ROI             │ 2.86x   │ 4.54x       │
└─────────────────────────────────────────┘

KEY TAKEAWAYS
• Q4 preparation starts in October
• Hourly monitoring on peak days is essential
• Brand defense during Q4 is critical
```

## Portfolio Formats

### Option 1: PDF Portfolio (Easiest)
- Use Google Docs or Canva
- Export as PDF
- Upload to Google Drive
- Share link in applications

### Option 2: Notion Portfolio (Modern)
- Create Notion page
- Use portfolio template
- Share public link
- Easy to update

### Option 3: Website Portfolio (Best)
- Use Carrd.co ($9/year) or GitHub Pages (free)
- Professional domain
- SEO-friendly
- Shareable link

### Option 4: LinkedIn Featured Section
- Add case studies as featured content
- Upload PDF portfolio
- Link to GitHub repo
- Visible to recruiters

## Portfolio Checklist

Before sending your portfolio:

- [ ] All numbers are accurate and verifiable
- [ ] Client names are anonymized (or permission obtained)
- [ ] Before/after metrics are clear
- [ ] No spelling or grammar errors
- [ ] PDF is under 5MB
- [ ] Contact info is current
- [ ] LinkedIn URL is working
- [ ] Portfolio is tailored to the specific job
- [ ] Case studies are relevant to the role
- [ ] Certifications are up to date
