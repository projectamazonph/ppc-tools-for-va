# Decisions — PPC Tools for VA (Project-Level)

## Decision 1: Markdown-First Content

**Context:** PPC resources need to be accessible, version-controlled, and easy to contribute to.

**Decision:** All resources stored as Markdown files in a flat directory structure. No database or CMS required.

**Consequences:**
- ✅ Easy to browse, edit, and version control
- ✅ GitHub renders markdown natively
- ✅ PR-friendly for community contributions
- ⚠️ No built-in search — requires external tool

## Decision 2: Open Source (MIT)

**Context:** The target audience is Filipino VAs with limited budgets. Free access maximizes reach.

**Decision:** License under MIT — free to use, share, and adapt. No restrictions.

**Consequences:**
- ✅ Maximum distribution and adoption
- ✅ Community contributions possible
- ⚠️ No direct revenue from the tool itself
- ⚠️ Must maintain quality without financial incentive
